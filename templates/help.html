What is Join?

Join is a program for visualizing and managing workflows. These include the ability to delegate tasks, add subtasks and set priorities. It supports collaboration within the team, improves transparency and increases productivity and work quality.

How to use it

1.

Understand the basics: Get used to the app, look how the tasks are visualized on the board, how you can create new ones and how to set properties. Add some new contacts and set some categories.

2.

Start with a simple board: Just add some simple tasks to the board to get familiar with the concept. As you become more comfortable with Join, you can add even more specialized task and customize your board to fit your needs.

3.

Collaborate with your team: Encourage your team members to create tasks on the board, update the status of tasks and add subtasks. It will help everyone stay on the same page and improve communication within the team.
